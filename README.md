# FashionKu Database Project

Sistem Informasi Penjualan Toko Baju FashionKu - Proyek Basis Data Universitas Esa Unggul.

## Struktur Folder
- `src/ddl`: Skrip CREATE TABLE dan ALTER TABLE
- `src/dml`: Skrip INSERT, UPDATE, DELETE
- `src/procedures`: Stored procedures dan functions
- `src/triggers`: Trigger otomatis
- `src/indexing`: Skrip indexing dan optimasi

- `docs/ERD`: Gambar ERD resolusi tinggi
- `docs/laporan`: Laporan proyek akhir
- `docs/panduan`: Panduan instalasi dan penggunaan
- `docs/kamus_data`: Penjelasan atribut dan tipe data

## Cara Instalasi
1. Clone repository ini
2. Import file SQL ke database MySQL
3. Jalankan skrip sesuai urutan: DDL → DML → Procedures → Triggers
